## Marcado de letras
### Cursivas
Las palabras se ponen dentro de uno de estos dos carecteres _ ó *
Ejemplo: _Esta es con underscore_ *Esta es con asterisco*
### Negritas
Las palabras van dentro de un par de asteriscos
Ejm: **Negrita**
